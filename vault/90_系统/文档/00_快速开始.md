# 快速开始

## 1. 启动 kos-agent

Obsidian Desktop 需要先安装 Node.js 22.19+，再把 kos Companion 发布目录放到：

```text
<Vault>/.obsidian/plugins/kos-companion
```

启用插件后打开右侧 kos Agent。首次使用点击“配置模型”，填写 provider、model ID、API key；使用自定义中转时同时填写 Base URL 和 API 协议。

API key 由 kos-agent 保存在自身配置目录，不进入 Vault 笔记或 Obsidian 插件数据。

## 2. 检查系统

在 kos Agent 侧栏点击系统检查按钮，或在 Vault 根目录运行无模型检查：

```bash
kos-harness validate
```

## 3. 开始一天

```text
帮我开始今天的工作：检查待审核、进行中项目和任务，更新今日工作台，并告诉我最值得推进的三件事。
```

## 4. 收集和处理输入

```text
把这份材料作为输入源收集到 kos，并提取重点、形成摘要：<链接或内容>
```

输入源默认进入 `11_原材料/`。Agent 会按需继续形成摘录、摘要、研究和概念。

## 5. 形成知识

```text
围绕 <研究问题> 做研究，把结论、证据和不确定性写入 kos；必要时创建相关概念。
```

## 6. 推进项目

```text
把 <项目想法> 建成项目，明确目标、当前阶段和下一步任务，然后开始推进。
```

## 7. 建立个人操作画像

```text
根据这些测评结果、复盘或协作偏好更新我的个人操作画像：<内容>
```

这会创建 `25_个人操作画像/` 下的 draft。它只表示当前可验证的协作假设，不是人格诊断。
